# Pingo.to AI skills

AI skills for <a href="https://pingo.to" target="_blank">https://pingo.to</a>.

## Install

```bash
npx skills add pingo-to/skills
```

## Usage

1, Set your pingo API key to env var, or add to your shell rc file (e.g. .bashrc, .zshrc, etc.).

2, Use it:

```
$ claude
> /pingo what can I do with this skill?
...
> /pingo pin this file ./hello.txt
```
